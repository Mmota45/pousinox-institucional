# Social Post

Gere posts completos para redes sociais da Pousinox, prontos para publicação.

## Contexto Pousinox
- **Instagram:** @pousinox — B2B industrial, foco em autoridade técnica e cases
- **Público:** Engenheiros, arquitetos, construtoras, revendas (decisores 30-55 anos)
- **Tom:** Profissional, técnico mas visual, autoridade sem ser frio
- **Cores:** Azul escuro + prata + branco (identidade visual)
- **Diferencial:** Único fixador inox, laudo SENAI, fabricante direto

## 1. Definir o post
Pergunte ao usuário:
- Tema: Produto / Case / Educativo / Bastidor / Prova social / Promocional?
- Rede: Instagram (feed/reels/stories) / LinkedIn / Facebook?
- Objetivo: Engajamento / Alcance / Tráfego / Conversão?

Se o usuário não especificar, sugira 3 ideias baseadas em:
- Calendário (datas do setor: Feicon, Expo Revestir, Dia do Engenheiro)
- Conteúdo educativo (erros comuns, dicas de instalação)
- Cases recentes (projetos finalizados)

## 2. Gerar conteúdo completo

### Legenda
- **Hook** (primeira linha — para o scroll): pergunta, dado surpreendente ou afirmação forte
- **Corpo** (3-5 linhas): informação de valor, história curta ou explicação
- **CTA** (última linha): ação clara (comentar, salvar, clicar no link)
- **Max:** 2200 chars (Instagram), 700 chars ideal para engajamento

### Hashtags (20-30)
Organizar em 3 grupos:
- **Nicho** (5-10): #fixadorporcelanato #revestimentofachada #obrasemretrabalho
- **Setor** (5-10): #construcaocivil #arquitetura #engenharia #obraindustrial
- **Alcance** (5-10): #dicadeobra #materialdeconstrucao #inovacao

### Sugestão Visual
- Tipo: foto produto / antes-depois / carrossel educativo / vídeo curto / infográfico
- Descrição da arte/foto ideal
- Texto overlay sugerido (se aplicável)
- Proporção: 1:1 (feed), 9:16 (reels/stories), 1.91:1 (LinkedIn)

### Horário sugerido
- Instagram B2B: Ter-Qui 7h-8h ou 17h-18h
- LinkedIn: Ter-Qui 8h-10h
- Evitar: Segunda manhã, Sexta tarde, finais de semana

## 3. Variações
Gere 3 variações de hook para teste:
- V1: Pergunta
- V2: Dado/número
- V3: Afirmação provocativa

## 5. Carrossel (se aplicável)
Para posts educativos, gere slides:
- Slide 1: Capa com headline forte
- Slides 2-8: 1 ponto por slide, visual limpo
- Slide final: CTA + contato

## 6. Regras
- Nunca usar emojis excessivos (max 3-4 por post, estratégicos)
- Linguagem profissional — não é influencer, é fabricante
- Sempre ter CTA (nem que seja "Salve este post")
- Dados concretos > frases genéricas
- Adaptar para o segmento se mencionado
