# Revisar SEO

Analise conteúdo e sugira melhorias de SEO.

## 1. Receber conteúdo
- Use o conteúdo da conversa atual (se veio de `/gerar-conteudo`)
- Ou peça o texto/URL ao usuário

## 2. Análise

### Título e Meta
- H1 presente? Max 60 chars?
- Meta description presente? 150-160 chars?
- Keyword principal no H1 e meta?

### Estrutura
- Hierarquia de headings correta (H1 → H2 → H3)?
- Parágrafos curtos (max 3-4 linhas)?
- Listas/tabelas para escaneabilidade?

### Keywords
- Keyword principal: densidade entre 1-2%?
- Keywords secundárias presentes nos H2s?
- Keyword no primeiro parágrafo?
- Variações/sinônimos usados?

### Links e CTAs
- Links internos para outras páginas do site?
- CTAs claros (WhatsApp, orçamento, contato)?
- Alt text em imagens sugerido?

### Técnico
- Schema markup sugerido (FAQ, Product, Organization)?
- Open Graph tags recomendadas?
- Canonical URL definida?
