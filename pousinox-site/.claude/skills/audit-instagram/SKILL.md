# Audit Instagram

Analise um perfil Instagram (próprio ou de cliente) e gere relatório com diagnóstico e recomendações práticas.

## 1. Coleta de dados

### Via prints/screenshots
Peça ao usuário:
- Print da bio (nome, descrição, link, destaques)
- Print do grid (últimos 9-12 posts)
- Print de 2-3 posts com métricas (curtidas, comentários)
- Print dos insights (alcance, impressões, seguidores) — se disponível

### Via URL (quando possível)
- Buscar perfil público e extrair dados visíveis
- Usar ctx_fetch_and_index se URL acessível

## 2. Análise — 7 Pilares

### Pilar 1: Bio e Primeira Impressão
| Critério | Ideal | Verificar |
|---|---|---|
| Nome de usuário | Memorável, sem números | Fácil de buscar? |
| Nome do perfil | Keyword + marca | Aparece em buscas? |
| Bio (150 chars) | Proposta de valor clara | Responde "o que faz, pra quem, resultado"? |
| CTA na bio | Ação clara | Link funcional? Linktree/direto? |
| Foto de perfil | Rosto ou logo nítido | Reconhecível em 32px? |
| Destaques | Organizados por tema | Capas padronizadas? Atualizados? |

### Pilar 2: Grid Visual
| Critério | Ideal | Verificar |
|---|---|---|
| Coerência visual | Paleta consistente | Max 3 cores dominantes? |
| Mix de formatos | Carrossel + Reels + Estático | Equilíbrio ou monotonia? |
| Thumbnails de Reels | Capa com texto | Clicável no grid? |
| Espaçamento visual | Respiro entre posts densos | Não sobrecarregado? |

### Pilar 3: Conteúdo e Copy
| Critério | Ideal | Verificar |
|---|---|---|
| Hook (1ª linha) | Gera curiosidade/parada | Cortaria o scroll? |
| Formato das legendas | Curtas OU longas com quebras | Parágrafos gigantes? |
| CTAs nos posts | Toda legenda tem ação | "Salve", "Comente", "Link na bio"? |
| Hashtags | 5-15 relevantes | Mix: nicho + volume? |
| Frequência | Min 3-5x/semana | Consistência? |

### Pilar 4: Engajamento
| Métrica | Bom | Ótimo | Verificar |
|---|---|---|---|
| Taxa engajamento | >1% | >3% | (curtidas+comentários)/seguidores |
| Comentários/post | >5 | >20 | Qualidade (conversa vs emoji)? |
| Saves/post | >2% reach | >5% | Indica valor percebido |
| Shares/post | >1% reach | >3% | Indica viralidade |

### Pilar 5: Posicionamento
- Nicho claro ou genérico?
- Diferencial visível em 5 segundos?
- Tom de voz consistente?
- Público-alvo evidente pelo conteúdo?
- Concorrentes diretos: como se diferencia?

### Pilar 6: Funil e Conversão
- Bio → Link → Página de destino: fluxo lógico?
- Stories com CTA para produto/serviço?
- Destaques funcionam como "mini-site"?
- Mix conteúdo: topo (atração) / meio (conexão) / fundo (conversão)?

### Pilar 7: Técnico
- Horários de publicação vs atividade do público
- Uso de Reels (alcance orgânico)
- Alt text em imagens?
- Geotag em posts locais?
