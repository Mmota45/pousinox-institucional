# Carrossel

Crie carrosséis profissionais para Instagram ou LinkedIn. Estrutura de slides + copy + design (Canva MCP ou manual).

## 1. Briefing

Pergunte ao usuário:
- **Tema:** Sobre o que é o carrossel?
- **Objetivo:** Educar / Vender / Engajar / Autoridade?
- **Plataforma:** Instagram (1080×1080 ou 1080×1350) / LinkedIn (1280×720)?
- **Slides:** Quantos? (ideal: 7-10 para Instagram, 5-8 para LinkedIn)
- **CTA final:** O que a pessoa deve fazer? (salvar, comentar, clicar, comprar)
- **Tom:** Técnico / Casual / Provocativo / Educativo?

## 2. Frameworks de estrutura

### Framework AIDA (vendas)
```
Slide 1: ATENÇÃO — Hook forte (pergunta/dado chocante)
Slide 2: INTERESSE — Contexto do problema
Slide 3-6: DESEJO — Solução passo a passo
Slide 7: AÇÃO — CTA claro
```

### Framework Lista (educativo)
```
Slide 1: Título "X [coisas] para [resultado]"
Slide 2-8: Um item por slide (número + título + 1 linha)
Slide 9: Resumo visual
Slide 10: CTA + "Salve para consultar depois"
```

### Framework Antes/Depois (transformação)
```
Slide 1: Hook "De [situação ruim] para [situação boa]"
Slide 2: O ANTES (dor, problema)
Slide 3: O PONTO DE VIRADA
Slide 4-7: O PROCESSO (passos da transformação)
Slide 8: O DEPOIS (resultado)
Slide 9: Prova (dados, foto, depoimento)
Slide 10: CTA
```

### Framework Mito vs Verdade (autoridade)
```
Slide 1: "X mitos sobre [tema] que te prejudicam"
Slide 2-8: ❌ Mito / ✅ Verdade (1 por slide)
Slide 9: Resumo
Slide 10: CTA + posicionamento
```

### Framework Storytelling (conexão)
```
Slide 1: Hook emocional
Slide 2: Situação inicial (identificação)
Slide 3: Conflito/desafio
Slide 4-6: Jornada/aprendizado
Slide 7: Resolução
Slide 8: Lição aplicável
Slide 9: CTA
```

## 3. Regras de copy por slide

### Slide 1 (Capa) — DECISIVO
- Max 8 palavras no título principal
- Subtítulo opcional (1 linha)
- Deve funcionar SOZINHO no feed (sem contexto)
- Gerar curiosidade ou identificação

Hooks que funcionam:
- "X erros que [público] comete em [tema]"
- "Como [resultado desejado] em [prazo/condição]"
- "Pare de [ação errada comum]"
- "[Número] + [promessa específica]"
- "O que ninguém te conta sobre [tema]"

### Slides internos (2 a N-1)
- Máximo 40 palavras por slide
- 1 ideia por slide (nunca 2)
- Hierarquia: Título bold → Subtexto leve
- Usar números, bullets ou ícones para escanear
- Progressão lógica (cada slide leva ao próximo)

### Slide final (CTA)
- Ação clara e única
- Urgência ou benefício de agir agora
- Perfil/logo visível
- Exemplos: "Salve este post ★", "Comente [X]", "Link na bio"

## 4. Design — Identidade Pousinox

### Paleta para carrosséis
```
Fundo: #1a2332 (azul escuro) ou #ffffff (branco)
Texto principal: #ffffff ou #1a2332 (contraste com fundo)
Accent: #2563eb (destaques, números, ícones)
Secundário: #b8c4d0 (subtextos, linhas)
```

### Tipografia
```
Títulos: Inter Bold, 32-48px
Subtítulos: Inter SemiBold, 24-28px
Corpo: Inter Regular, 18-22px
Caption/fonte: Inter Regular, 14-16px
```

### Layout por tipo de slide
```
┌─────────────────────┐   ┌─────────────────────┐
│                     │   │      [ícone]         │
│   TÍTULO BOLD       │   │                     │
│   subtítulo         │   │   Título do ponto   │
│                     │   │   Texto explicativo  │
│        [capa]       │   │   curto aqui        │
│                     │   │                     │
│   @pousinox         │   │   @pousinox    3/10 │
└─────────────────────┘   └─────────────────────┘

┌─────────────────────┐   ┌─────────────────────┐
│  ❌ MITO            │   │                     │
│  "texto do mito"    │   │  ★ SALVE ESTE POST  │
│                     │   │                     │
│  ✅ VERDADE         │   │  Quer [resultado]?  │
│  "texto verdadeiro" │   │  [CTA específico]   │
│                     │   │                     │
│   @pousinox    4/10 │   │  @pousinox   [logo] │
└─────────────────────┘   └─────────────────────┘
```

### Regras visuais
- Margem interna: 60px (safe zone Instagram)
- Número do slide: canto inferior direito (ex: 3/10)
- Logo/@ no rodapé de todos os slides
- Máximo 2 fontes por carrossel
- Consistência de alinhamento (centro ou esquerda, nunca misturar)
- Imagens: usar como fundo com overlay escuro (70% opacity)

## 5. Geração via Canva MCP

Se Canva MCP disponível:
```
1. Criar design: formato 1080×1080 (Instagram) ou 1080×1350 (vertical)
2. Aplicar template com paleta Pousinox
3. Para cada slide:
   - Definir background (cor sólida ou imagem+overlay)
   - Adicionar texto com hierarquia
   - Posicionar elementos (ícone, número, logo)
4. Exportar como PDF ou imagens individuais
```

Se Canva MCP não disponível:
- Entregar Markdown com texto de cada slide
- Indicar layout sugerido (diagrama ASCII)
- Listar assets necessários (fotos, ícones)
