## Recomendação de importação

### Fase 1 — Quick wins (alta conversão)
- CNAEs 25.xx (produtos de metal) + 28.62 (máquinas para alimentos) + 33.xx (manutenção industrial)
- Filtro geográfico: MG, SP, PR, SC, RS
- Estimativa: ~50K-100K CNPJs

### Fase 2 — Indústria alimentícia
- CNAEs 10.xx + 11.xx
- Sobreposição com prospects existentes (açougue/frigorífico são varejo, aqui é indústria)
- Estimativa: ~30K-60K CNPJs

### Fase 3 — Química/Farmacêutica/Metalurgia
- CNAEs 20.xx + 21.xx + 24.xx
- Ticket médio mais alto, ciclo de venda mais longo
- Estimativa: ~20K-40K CNPJs

### Fonte de dados
- Receita Federal (base CNPJ aberta) — mesma fonte da base atual
- Filtrar por situação cadastral ativa + porte (excluir MEI)
- Campos: CNPJ, razão social, nome fantasia, CNAE principal, UF, município, porte

### Segmento sugerido na importação
Atribuir `segmento = 'Indústria'` para todos os CNAEs acima, ou subcategorias:
- `Indústria Alimentícia` (10.xx, 11.xx)
- `Indústria Metalúrgica` (24.xx, 25.xx)
- `Indústria de Equipamentos` (28.xx)
- `Indústria Química/Farmacêutica` (20.xx, 21.xx)
- `Manutenção Industrial` (33.xx)
